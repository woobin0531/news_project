# AI 에이전트 기반 개인 맞춤 뉴스 서비스 (news_project)

## 📰 프로젝트 소개

본 프로젝트는 **AI Agent 기술**을 활용하여, 사용자의 음성 명령/자연어 요청을 인식해 주요 키워드를 추출하고, 그에 맞춘 뉴스를 실시간 크롤링·요약·추천해주는 **개인화 뉴스 서비스**를 목표로 합니다.

- **복합 키워드 이중 쿼리** 구조를 이용해 매칭 정확도를 크게 향상시켰습니다.
- STT, LLM, 자연어 해석, 크롤링, 요약, TTS 출력까지 전체 파이프라인을 자동화했습니다.
- React 기반의 UI/UX로 직관적인 음성 조작 인터페이스를 제공합니다.

## 👀 배경 및 필요성

- **정보 과잉 해결:** 정보 과잉 시대에 관심사에 맞는 뉴스 필터링의 한계와 난이도를 극복하고자 했습니다.
- **맥락 기반 검색:** 단순 키워드 매칭 이상의 복합 질의를 지원하여, 사용자의 실제 요구 의도와 맥락을 반영합니다.

<summary><b>🏗️ 시스템 아키텍처 및 기술 스택</b></summary>

- **Voice Interface:** Whisper 기반 STT, OpenAI TTS (사용자 음성 → 텍스트 / 텍스트 → 음성 변환)
- **AI Agent:** LangChain 기반 명령 해석, GPT-4o/LLM 다단계 요약 및 프롬프트 엔지니어링
- **News Crawler:** Naver News API, BeautifulSoup, 언론사별 구조 대응 파싱
- **통합 백엔드:** FastAPI (Python), 비동기 데이터 핸들러
- **프론트엔드:** React, HTML, JavaScript, CSS 기반 (반응형 웹)
- **안정성:** 전체 통합 테스트 시나리오 성공률 **90% 이상** 달성

</details>

- **뉴스 엔진:** 실시간 뉴스 검색/크롤링/정제, 멀티 기사 통합 요약
- **인터랙티브 UI:** TTS(텍스트→음성), 맞춤형 UI(음성 조작/결과 브리핑/애니메이션 포함)

## ✨ 기술·성과 요약

- **정확도 18% 향상:** 이중 쿼리 구조 도입으로 뉴스 필터링 정확도를 높였습니다. (내부 테스트 기준)
- **시스템 안정성:** 통합 기능 모듈화 및 테스트 성공률 **90% 이상**을 달성했습니다.
- **UX 최적화:** 자동 프롬프트 조정, 다단계 요약, 끊김 없는 화면 전환을 통해 사용자 만족도 **88%**를 기록했습니다.
